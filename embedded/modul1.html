Modul 1

Input dan Output Data Digital

Tujuan

  • Mengatur waktu kedip lampu LED
  • Membaca status sebuah switch

A. Mengatur waktu kedip lampu LED


Peralatan yang diperlukan:
  • Board Arduino Uno
  • LED
  • Resistor 220 ohm
  • Breadboard
Tugas
  1. Modifikasi agar program tersebut dalam 1 detik, LED menyala dalam keadaan ON selama 2/3 detik dan LED dalam keadaan OFF selala 1/3 detik.
  2. Menambahkan 4 buah LED, sambungkan pada pin digital lain dengan cara yang sama. Susun kode program sehingga LED menyala secara bergiliran
  3. Buatlah animasi sederhana dengan 5 buah lampu LED


Kode Program
int ledIjo = 13;
int ledMerah = 12;
int ledKuning = 11;

void setup() {
 
  // put your setup code here, to run once:
  pinMode(ledIjo, OUTPUT);
  pinMode(ledMerah, OUTPUT);
  pinMode(ledKuning, OUTPUT);
}

void loop() {
  // put your main code here, to run repeatedly:
  digitalWrite(ledIjo, HIGH);
  delay(1000);
  digitalWrite(ledIjo, LOW);
  delay(1000);
  digitalWrite(ledKuning, HIGH);
  delay(1000);
  digitalWrite(ledKuning, LOW);
  delay(1000);
  digitalWrite(ledMerah, HIGH);
  delay(1000);
  digitalWrite(ledMerah, LOW);
  delay(1000);
}

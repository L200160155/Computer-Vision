Modul 3

Display 7-Segment

Tujuan

  • Menampilkan angka dengan komponen display 7-segment

Peralatan yang diperlukan:

  • Board Arduino
  • Display 7-segment
  • Resistor 1K ohm
  • Breadboard

Tugas
  1. Tambahkan rangkaian potensiometer pada rangkaian 7-Segment kemudian susun program untuk menampilkan angka sesuai dengan posisi putar potensiometer. Bedakan posisi potensiometer menjadi 10 bagian, 0-100, 101-200,..., 901-1023

Kode Program
      int potPin = A0;
      int motorPin = 9;
      int potValue = 0;
      int motorValue = 0;

      void setup() {
        // put your setup code here, to run once:
        
      }

      void loop() {
        // put your main code here, to run repeatedly:
        potValue = analogRead(potPin);
        motorValue = map(potValue, 0, 1023, 0, 255);
        analogWrite(motorPin, motorValue);
        delay(2); 
      }

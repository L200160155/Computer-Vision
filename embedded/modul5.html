Modul 5

Motor DC

Tujuan

  • Mengatur kecepatan putar motor DC

Peralatan yang diperlukan:

  • Board Arduino
  • Motor DC
  • Resistor 10K dan 2.2K ohm
  • Breadboard
  • Potensiometer
  • Transistor

Tugas
  1. Gunakan 7 buah LED sebagai indikator kecepatan putar motor DC, gambarkan rangkaian dan susun kode programnya.

Kode Program
      int potPin = A0;
      int motorPin = 9;
      int potValue = 0;
      int motorValue = 0;
      int led = 0;
      int val[] = {B10111111, B11011111, B11101111, B11110111, B11111011, B11111101, B11111110};

      void setup() {
        // put your setup code here, to run once:
        DDRD = B11111111;
      }

      void loop() {
        // put your main code here, to run repeatedly:
        potValue = analogRead(potPin);
        motorValue = map(potValue, 0, 1023, 0, 255);
        analogWrite(motorPin, motorValue);
        led = map(potValue, 0, 1023, 0, 7);
        PORTD = val[led];
        delay(2);
      }
